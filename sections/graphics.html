▶
TechTypeBest forLimit
SVGVector DOMCharts, diagrams, icons (<5k elements)Slow with many elements
Canvas 2DRaster JS APIGames, particles, large datasetsNot accessible per-element

<1k elements → SVG; 1k–100k → Canvas; 100k+ or 3D → WebGL/PixiJS. Use D3's .join() API (v5+) over manual enter/exit.
